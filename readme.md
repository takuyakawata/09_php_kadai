## 工夫した点

  - ログイン機能の追加

  - ホーム画面のメニューバーがどの画面になっても残る（phpをつかって）

  - APIのデータを受け取って、本の検索ができる。

  - カーリルのAPIを使って図書館検索（リンクで飛ぶだけ）

  - 登録したい本のデータを直接入力することができる(画像なし)

  -　ユーザーごとに登録した内容の表示・変更・削除など編集ができる
     ユーザーごとに登録段階で、番号を振ることで裏では番号によって管理ができるようになっている
